# Roundtable 圆桌工作流

Roundtable（圆桌会议）是未来多角色工作流。用户可以选择多个角色参与任务讨论，系统可以让角色分别发言、互相质疑、由主持人总结，最后给用户对齐后的方案。

## 早期形态

早期不实现桌面动画圆桌。最早可以从文本会议模式开始，让用户看到每个角色的观点和主持人的总结。

## 后续能力

- 桌面气泡讨论。
- 角色争吵。
- 投票。
- 对齐方案。
- 角色融合。

## 体验原则

圆桌会议必须可选，不能影响普通提醒用户的低门槛体验。普通用户只想要喝水、休息、学习或工作提醒时，不应该被多角色工作流打扰。

V0.0 不实现多角色对话或圆桌会议。
