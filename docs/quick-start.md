# 快速开始

CodePet V0.1 是桌宠壳子 MVP，用于验证小尺寸桌面窗口、拖动、置顶、系统托盘、关闭隐藏到托盘和状态气泡展示。

## 启动开发版

```bash
cd apps/desktop
pnpm install
pnpm tauri dev
```

启动后你应该看到一个小尺寸 CodePet 桌面窗口，默认靠近桌面右下角。你可以拖动窗口、切换状态、切换置顶，也可以在系统托盘里显示、隐藏或退出应用。

## V0.1 验收

- 桌面能看到 CodePet 小窗口。
- 窗口可以拖动。
- 点击“待机 / 专注 / 提醒 / 完成 / 警告”后，气泡文案会变化。
- 点击“窗口置顶 / 取消置顶”后，窗口置顶状态会切换。
- 系统托盘存在，菜单包含显示、隐藏、切换置顶、退出。
- 点击窗口关闭按钮或触发关闭时，应用隐藏到托盘，不直接退出。
- 点击托盘“退出”后应用真正退出。

## 当前不要期待的功能

- 没有提醒调度。
- 没有桌宠动画。
- 没有 SQLite 数据库。
- 没有 Ollama、本地模型或云模型调用。
- 没有 Codex / Cursor / Claude Code 监控。
- 没有记忆、Skill 执行或多角色圆桌会议。
