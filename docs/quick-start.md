# 快速开始

CodePet V0.3 是 Ollama 本地模型接入版本，用于验证本地提醒、本地提示音、本机 Ollama 检测、本地聊天和 AI 提醒文案生成。

## 启动开发版

```bash
cd apps/desktop
pnpm install
pnpm tauri dev
```

启动后你应该看到 CodePet 桌面窗口。你可以拖动窗口、切换状态、切换置顶，也可以在系统托盘里显示、隐藏或退出应用。

## 创建第一个提醒

1. 打开“本地提醒”。
2. 填写标题，例如“站起来活动一下”。
3. 选择提醒类型和模式。
4. 间隔提醒填写分钟数，固定时间提醒选择每天触发时间。
5. 可选填写提醒文案。
6. 选择是否播放提示音，以及使用哪个提示音。
7. 点击“创建提醒”。

提醒到时间后，CodePet 会切换到 `reminding` 状态，气泡显示提醒文案，并在启用提示音时播放本地提示音。你可以选择“完成”“稍后提醒”或“忽略”。

## 本地 AI 设置

1. 确认已经安装并启动 Ollama。
2. 打开“本地 AI 设置”。
3. 默认 API 地址是 `http://localhost:11434/api`。
4. 点击“重新检测”，查看本地模型列表。
5. 如果没有模型，请在终端执行推荐命令，例如：

```bash
ollama pull qwen3:0.6b
```

6. 在模型下拉框中选择默认模型。
7. 勾选“启用本地 AI”，点击“保存本地 AI 配置”。

## 测试本地聊天

打开“本地聊天”，输入一句话并点击“发送”。发送时桌宠会进入 `thinking` 状态，模型回复后气泡会显示简短回复。

## AI 提醒文案

在创建或编辑提醒时，可以点击“让 AI 帮我生成提醒文案”，也可以选择更温柔、更严格、更可爱或更简短。生成结果只会填入文案框，仍然需要你点击保存提醒。

## 当前不要期待的功能

- 没有桌宠动画。
- 没有 Codex / Cursor / Claude Code 监控。
- 没有通用命令监控。
- 没有 AI 语音或本地 TTS。
- 没有记忆、Skill 执行或多角色圆桌会议。
