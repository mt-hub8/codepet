# 快速开始

CodePet V0.3.5 是本地产品 UI 减负与首页重构版本，在保留 V0.1 桌宠、V0.2 提醒、V0.3 Ollama 本地 AI 的前提下，提供更轻量的默认首页体验。

## 启动开发版

```bash
cd apps/desktop
pnpm install
pnpm tauri dev
```

启动后你会看到 CodePet 本地工作面板。窗口可拖动、可置顶，关闭后会隐藏到系统托盘。

## 使用首页

默认首页包含以下区域：

1. **顶部问候区**：根据时间显示早安 / 下午好 / 晚上好，右上角展示桌宠在线状态与 Ollama 连接状态。
2. **今日提醒**：展示最多 3 条已启用提醒；若无提醒，会提示你先创建喝水或休息提醒。
3. **自定义角色**：展示 4 个轻量角色预设卡片，点击进入角色页查看详情占位。
4. **最近任务状态**：当前为 V0.4 占位空状态，不会启动任何外部命令。

## 进入提醒管理

1. 点击左侧导航「提醒」，或首页「今日提醒」区的「查看全部」。
2. 填写标题、类型、模式和文案。
3. 选择提示音并创建提醒。
4. 提醒到时间后，顶部会出现提醒横幅，桌宠状态切换为 `reminding`。

## 使用本地 AI

1. 点击左侧导航「本地 AI」。
2. 在「AI 设置」中检测 Ollama、选择模型并启用本地 AI。
3. 切换到「本地聊天」测试模型回复。
4. 在提醒表单中可使用 AI 生成提醒文案（需先启用本地 AI）。

## 查看角色卡片

点击左侧导航「角色」，可查看 4 个默认角色预设。本阶段仅为 UI 展示，不含记忆、Skill 或完整编辑器。

## 任务监控占位

点击「任务监控」可看到 V0.4 示例占位。当前不会监控任何 CLI 或外部命令。

## 设置与桌宠调试

在「设置」页可以切换窗口置顶，并使用桌宠状态测试面板手动切换状态。

## 当前不要期待的功能

- 没有完整角色工作室。
- 没有 Petdex 素材导入。
- 没有 Codex / Cursor / Claude Code 监控。
- 没有通用命令监控。
- 没有 AI 语音或本地 TTS。
- 没有记忆、Skill 执行或多角色圆桌会议。
