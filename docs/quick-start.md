# 快速开始

CodePet V0.0 是工程骨架版本，只用于验证桌面应用可以启动、目录结构清晰、文档方向明确。

## 启动开发版

```bash
cd apps/desktop
pnpm install
pnpm tauri dev
```

启动后你应该看到一个基础 CodePet 桌面窗口，显示当前阶段和预留模块边界。

## 当前不要期待的功能

- 没有提醒调度。
- 没有桌宠动画。
- 没有系统托盘。
- 没有 SQLite 数据库。
- 没有 Ollama、本地模型或云模型调用。
- 没有 Codex / Cursor / Claude Code 监控。
- 没有记忆、Skill 执行或多角色圆桌会议。
