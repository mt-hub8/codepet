# 快速开始

CodePet V0.2 是本地提醒系统版本，用于验证从创建提醒、保存到本地、到时间触发、桌宠气泡提醒、播放提示音、记录历史的完整闭环。

## 启动开发版

```bash
cd apps/desktop
pnpm install
pnpm tauri dev
```

启动后你应该看到 CodePet 桌面窗口，默认靠近桌面右下角。你可以拖动窗口、切换状态、切换置顶，也可以在系统托盘里显示、隐藏或退出应用。

## 创建第一个提醒

1. 在“本地提醒”面板中填写标题，例如“站起来活动一下”。
2. 选择提醒类型和模式。
3. 如果选择“间隔”，填写间隔分钟；如果选择“固定时间”，选择每天触发时间。
4. 可选填写提醒文案。
5. 选择是否播放提示音，以及使用哪个提示音。
6. 点击“创建提醒”。

提醒到时间后，CodePet 会切换到 `reminding` 状态，气泡显示提醒文案，并在启用提示音时播放本地提示音。你可以选择“完成”“稍后提醒”或“忽略”。

## 提示音

- V0.2 内置一个默认提示音，由浏览器音频能力生成轻量提示音。
- 你可以导入 `mp3`、`wav`、`ogg` 文件作为自定义提示音。
- 导入后文件会复制到 CodePet 应用数据目录，不依赖原文件路径。
- 自定义提示音可以试听和删除，内置提示音不可删除。

## V0.2 验收

- 桌面能看到 CodePet 小窗口。
- 窗口可以拖动。
- 点击“待机 / 专注 / 提醒 / 完成 / 警告”后，气泡文案会变化。
- 点击“窗口置顶 / 取消置顶”后，窗口置顶状态会切换。
- 系统托盘存在，菜单包含显示、隐藏、切换置顶、退出。
- 点击窗口关闭按钮或触发关闭时，应用隐藏到托盘，不直接退出。
- 点击托盘“退出”后应用真正退出。
- 首次启动后有默认提醒模板。
- 重启应用后提醒配置仍然存在。
- 提醒触发时可以播放默认提示音。
- 用户可以导入、试听、删除自定义提示音，并为不同提醒选择不同提示音。

## 当前不要期待的功能

- 没有桌宠动画。
- 没有 Ollama、本地模型或云模型调用。
- 没有 Codex / Cursor / Claude Code 监控。
- 没有 AI 语音、记忆、Skill 执行或多角色圆桌会议。
