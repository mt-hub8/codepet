# 截图资源

本目录存放 CodePet Public Beta 展示用截图，供 README 与文档引用。

## 需要补充的文件

请将以下 PNG 截图放入本目录（建议宽度 1280px 左右，统一浅色主题）：

| 文件名 | 内容 |
|--------|------|
| `home.png` | 首页（问候、工作状态、提醒摘要） |
| `onboarding.png` | 新手引导欢迎页 |
| `reminders.png` | 提醒管理页 |
| `local-ai.png` | 本地 AI 设置页 |
| `command-monitor.png` | 任务监控页 |
| `dependency-doctor.png` | 依赖诊断页 |
| `pet-library.png` | 宠物素材库 / 角色页 |

## 截图规范

- 不要包含真实 API Key、Token、密码
- 路径脱敏（避免完整用户名目录）
- 命令日志脱敏
- 宠物素材使用有授权或项目自带占位素材
- 使用 Public Beta 版本界面截图

详细清单见 [docs/demo.md](../demo.md)。
