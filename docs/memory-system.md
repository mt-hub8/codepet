# 记忆系统

CodePet 的记忆默认本地保存。V0.0 不实现真实记忆检索，只定义边界和隐私原则。

## 记忆类型

记忆后续可以分为：

- 用户全局记忆。
- 角色私有记忆。
- 共享记忆空间。
- 任务记忆。
- 对话记忆。
- 工作流记忆。

## 隔离原则

角色记忆默认隔离。共享记忆必须用户显式开启，不能默认把一个角色的记忆暴露给所有角色。

## 角色融合

角色融合时不覆盖原角色，而是创建新角色。新角色可以引用父角色的记忆摘要，原始记忆不应该被默认复制。

## 用户控制

用户必须可以查看、删除、导出记忆。记忆系统的任何自动化行为都应该可解释、可关闭。

## 后续技术方向

后续语义记忆可以使用 SQLite + 向量检索，但 V0.0 不实现 SQLite 表、向量索引或语义检索。
