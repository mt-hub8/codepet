# 发布策略

CodePet 早期优先支持 Windows，后续再支持 macOS / Linux。

## 阶段目标

- V0.0 不发布安装包，只提供源码骨架。
- V0.6 目标是 Windows 安装包。
- 后续通过 GitHub Release 发布正式安装包。
- macOS / Linux 安装包在后续版本预留。

## Release 内容

每次 Release 应该包含：

- 安装包。
- release notes（发布说明）。
- changelog（变更日志）。
- 已知问题。
- 升级说明。

## 自动更新

V0.0 不实现自动更新。自动更新后续放到 V1.1。
