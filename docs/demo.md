# CodePet 演示指南

本文供维护者录制 Demo、补充 README 截图与对外展示使用。

---

## 3 分钟 Demo 路线

适合短视频或快速介绍。

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 打开 CodePet | 展示桌宠与主窗口 |
| 2 | 看首页和桌宠状态 | 问候、今日提醒、工作状态摘要 |
| 3 | 创建一个喝水提醒 | 提醒页 → 新建 → 60 分钟间隔 |
| 4 | 打开本地 AI 设置 | 展示 Ollama 地址与检测按钮 |
| 5 | 展示 Ollama 检测 | 有/无 Ollama 均可，强调友好提示 |
| 6 | 创建一个 `pnpm build` 监控任务 | 仅创建草稿或运行演示项目，**不要自动执行危险命令** |
| 7 | 展示成功 / 失败状态 | 桌宠气泡 + 首页任务摘要 |
| 8 | 打开依赖诊断页 | 展示检测项与脱敏诊断信息 |
| 9 | 展示宠物素材库 | 默认桌宠 + 导入入口 |
| 10 | 展示宠物孵化向导 | 生成并复制提示词（不上传） |

## 10 分钟完整 Demo 路线

适合 README 录屏、发布会或深度介绍。

1. **新手引导** — 8 步概览，演示跳过与完成
2. **提醒系统** — 默认提醒、自定义提示音、提醒历史
3. **本地 AI** — Ollama 检测、选模型、本地聊天（可选）
4. **命令监控** — 通用命令、stdout/stderr、取消与重跑
5. **Agent CLI** — 依赖诊断中 Codex / Cursor / Claude Code 状态
6. **等待确认 / 无输出** — 说明检测逻辑与限制（可用模拟输出）
7. **宠物导入** — `pet.json` + spritesheet 流程
8. **工作状态中心** — 首页摘要 + 详情页
9. **基础记忆** — 设置页查看与删除
10. **隐私说明** — 本地数据目录、诊断脱敏、不上传承诺

## Demo 注意事项

- **不要**展示真实 API Key、Token、密码
- **不要**展示私人路径（如完整 `C:\Users\真实用户名\...`）
- **不要**展示私人项目代码或敏感仓库名
- 命令日志截图需**脱敏**
- 宠物素材不要使用**无授权**的第三方角色
- 远程 Ollama 演示时说明数据可能离开本机
- Agent 任务需**用户主动点击启动**，Demo 中说明 CodePet 不会自动确认权限

## 推荐演示环境

- Windows 10 / 11
- CodePet v1.0 Public Beta 安装包或 `pnpm tauri dev`
- 可选：本机 Ollama + `qwen3:0.6b`
- 演示用空目录或公开示例仓库作为命令监控工作目录
