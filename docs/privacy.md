# 隐私说明

CodePet 默认本地运行，默认不上传用户代码、任务内容、聊天记录或角色记忆。

V0.2 的提醒配置、提醒历史和提示音配置默认保存在本机 SQLite 数据库中。用户导入的提示音文件会复制到本机应用数据目录下的 `sounds` 目录，不上传到任何服务。

## 默认策略

- 默认不上传用户代码。
- 默认不上传用户任务内容。
- 默认不上传聊天记录。
- 默认不上传角色记忆。
- 默认不上传提醒配置。
- 默认不上传提醒历史。
- 默认不上传用户导入的提示音文件。
- 用户显式配置云 API 后才可能外发请求。

## 外部工具提醒

Codex / Cursor / Claude Code 等外部工具可能有自己的数据策略。CodePet 文档需要提醒用户自行确认这些工具的隐私政策、数据保留规则和命令执行行为。

## 记忆数据

角色记忆和共享记忆必须可查看、可删除、可导出。共享记忆必须由用户显式开启。

## 不要误提交

本地数据库、日志、模型文件不要误提交到 GitHub。请确认 `.gitignore` 覆盖：

- `*.sqlite`
- `*.db`
- `logs/`
- `models/`
- `sounds/`
- `*.log`
- `.env`

AI 语音不是 V0.2 功能。后续如果接入云端 TTS，必须明确告知用户哪些文本会发送到云端；如果接入本地 TTS，也需要说明模型文件体积、缓存位置和删除方式。
