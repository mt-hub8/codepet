# Roadmap

CodePet 从轻量桌宠骨架逐步扩展到本地 AI、Agent 监控与社区生态。历史版本 V0.0–V0.7 已完成，详见下方归档。

---

## V1.0 Public Beta Readiness（已完成）

**CodePet v1.0.0-beta.1 Public Beta** — 公开 Beta 发布准备与 Release Candidate。

- README 公开展示首屏、下载入口、隐私与限制说明
- Demo 文档、QA 清单、已知问题
- Issue / PR 模板、CONTRIBUTING、SECURITY
- 截图目录占位与 Release Notes 模板
- 版本号 `1.0.0`，标签 `v1.0.0-beta.1`，文档标注 **CodePet v1.0.0-beta.1 Public Beta**

---

## V1.1 Stability & Feedback Fixes

根据 Beta 反馈收敛：

- 修 bug、修 CI、修安装体验
- 兼容性改进（Windows 路径、权限、CLI 检测）
- 补充文档与截图
- 不堆新业务能力

---

## V1.5 Character Studio & Skill Lite

- 角色工作室（外观与人格配置）
- Skill Lite 与角色绑定
- 角色记忆边界与用户确认式 Skill 建议

---

## V2.0 Plugin / MCP / Multi-Agent Platform

- MCP 与插件系统
- 多 Agent 工作流
- 圆桌讨论与协作编排

---

## V3.0 Community Ecosystem

- 角色模板与 Skill 分享
- 宠物素材生态
- 在线素材目录（前提是版权与授权机制清晰）

## 明确不在近期范围

- 自动更新、代码签名（V1.1 仍不承诺）
- AI 语音 / TTS
- 应用内遥测
- 在线素材市场（V3.0 前提条件：版权机制）
